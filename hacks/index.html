OS 開発者向けドキュメント

Firefox OS を他の端末にポーティングしたり、Firefox OS 自体を書き換えて楽しみたい人向けのドキュメントは主に Mozilla Developer Network に揃っています。セミナーなどの講演資料は Wiki にまとめられているのでそちらも参考にしてください。

Firefox OS | Mozilla Developer Network


https://developer.mozilla.org/ja/docs/Mozilla/Firefox_OS
Firefox OS 開発者向けのドキュメントが多数用意されています。例えば次のようなドキュメントを参考にして開発を進めていってください。

  • Firefox OS 概要
  • Firefox OS のビルドとインストール
  • .userconfig ファイルをカスタマイズする
  • Build System Primer
  • Firefox OS のアーキテクチャ
  • Debugging on Firefox OS
  • Firefox OS の Security
  • Firefox OS の Performance
  • Porting Boot to Gecko

日本語になっていない記事や英語版に比べて古くなっている記事を見つけたらドキュメント翻訳のガイドラインを参考に日本語化にご協力いただければ幸いです。

Mozilla Mobile Partners


https://mobilepartners.mozilla.org/
Firefox OS を使用したデバイスを開発するキャリアやハードウェアメーカーなどのパートナー向けサイトです。Firefox OS の概要、ロードマップ、OS のビルドやポーティング、商用展開に向けてのガイドラインなどのドキュメントがまとめられています。パートナーはもちろん開発者個人にとっても有用なドキュメントや入り口になるかと思います。

Mozilla Hacks ブログ翻訳 | Mozilla Developer Street (modest)


https://dev.mozilla.jp/project/hacksmozillaorg/
Mozilla Hacks Blog では Firefox や Web 技術に関する最新情報が発信されていますが、ここではその記事をコミュニティの有志が翻訳しています。Firefox OS カテゴリの記事はもちろん、それ以外に面白い記事が多いので、未訳の良い記事があったらあなたもここに翻訳を掲載してください。

Firefox OS のリポジトリ (ソースコード)

  • Firefox OS メインリポジトリ: https://github.com/mozilla-b2g/B2G/
  • Gaia (ユーザインターフェイス層): https://github.com/mozilla-b2g/gaia
  • Gonk (Linux & HAL 層): https://github.com/mozilla-b2g/gonk-misc
  • 日本語 L10N ファイル: https://hg.mozilla.org/gaia-l10n/ja/

コミットログを追いたい場合は各リポジトリの RSS フィードを購読したり、Twitter の @mozb2g アカウントをフォローするなどしていただけます。

また、Firefox OS のコードをみんなで読んで知見を共有する Firefox OS コードリーディングではまとめサイト的な Wiki やソースコードディレクトリ単位でメモを残すディレクトリマップを用意しているのでそちらも参考にしてください。
