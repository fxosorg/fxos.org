au Fx0 導入から活用まで


/docs/fx0/
2014年12月にKDDIからauブランドで発売された、日本発のFirefox OS搭載スマートフォン「Fx0」について、使い方など関連情報を整理したページで、次のような情報を掲載しています。

  • Fx0を入手する
  • Fx0を使い始める
  • Fx0をアップデートする
  • 格安SIMで運用する
  • アプリを使う【実用編】
  • アプリを使う【エンタメ編】
  • 便利な機能
  • ロックスクリーンを使う
  • Fx0でクリエイトする
  • その他、関連情報など

Firefox OS ヘルプ（Mozilla Support）


https://support.mozilla.org/ja/products/firefox-os
Firefox OS のユーザ向けサポートサイトです、以下のようなトピック別にヘルプドキュメントが用意されています:

  • 基本を学ぼう: はじめに
  • プライバシーとセキュリティの設定
  • 遅い動作、クラッシュ、エラーメッセージの解決と他の問題
  • ツールバーのカスタマイズ、オプション、設定、環境設定
  • アプリのダウンロードとインストール
  • コミュニティのサポートを受ける

Firefox Marketplace


https://marketplace.firefox.com/
Web アプリケーションのマーケットプレイスです。Firefox OS はもちろん、Android や PC でも Firefox さえあれば同じアプリをインストールしてマルチデバイスで使えるように開発が進められています。

Firefox OS | Mozilla Developer Network


https://developer.mozilla.org/ja/docs/Mozilla/Firefox_OS
Firefox OS の開発者向けドキュメントサイトです。ユーザの皆さんでもご興味のある方はご覧ください。その他にアプリ開発者ドキュメントや Firefox OS 本体をハックする方向けのドキュメントもまとめています。
