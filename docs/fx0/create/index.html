Fx0でクリエイトする

【au Fx0 導入から活用まで】 > 【Fx0でクリエイトする】

ここでは、ギーク向けスマホと呼ばれるFx0の特徴である、クリエイティブな使い方の参考情報を紹介します。

アプリ作成ツールFramin

Framinは、スマホだけでアプリを開発できるツールです。画面をタップしたり、スマホをふったりといった動作で、画面を切り替えたり、アラームをセットするというように、アイデア次第で手軽にアプリを作成できます。

また、作成したアプリをロックスクリーンにしたり、他のユーザーと共有することもできます。

Framinを初期設定する

  • 1.「Framin」アイコンをタップ
  • 2. インストールを開始する
  • 3. 再度、「Framin」アイコンをタップ
  • 4. 表示された簡易チュートリアルを読む
  • 5. 利用規約に同意する
