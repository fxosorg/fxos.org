アプリ開発者向け ドキュメント

Firefox OS 関連のアプリ開発者向けドキュメント

Firefox OS アプリ (Open Web Apps) 開発者向けには、Marketplace や Mozilla Developer Network に豊富なドキュメントが用意されています。セミナーなどの講演資料は Wiki にまとめられているのでそちらも参考にしてください。

アプリ提供事業者向け Firefox OS セミナー


http://www.mozilla.jp/events/2015/seminar-0213/
2015 年 2 月に実施されたこのセミナーではアプリ開発の基本から Firefox Marketplace でのアプリ配信まで一通り解説されており、講演のスライド、ビデオ、レポートが公開されています。全体を概観するには先ずはここから見てみるのも良いかも知れません。

Developers | Firefox Marketplace


https://marketplace.firefox.com/developers/
Firefox Marketplace の開発者向けページです。個別のドキュメントは次に説明する MDN へのリンクになっているものも少なくないですが、開発者向けの基本的な情報が集められています。

ただしこちらのサイトは現在のところ国際化 (翻訳) に対応しておりませんので、日本語ドキュメントが欲しい方は次に紹介する MDN のドキュメントをご覧ください。

  • Design Fundamentals (デザインの基本)
  • UI Guideline (UI ガイドライン)
  • Firefox OS Quick Start (クイックスタート)
  • Payments (課金システム)
  • Marketplace Submission (Marketplace への登録)

Apps | MDN（Mozilla Developer Network）


https://developer.mozilla.org/ja/docs/Apps
アプリ開発者向けのドキュメントとしてチュートリアルや手順解説、API リファレンスなどが用意されています。

  • Web アプリの基本
    • チュートリアル
    • アプリのテンプレートを使う
    • マニフェスト (アプリ情報) ファイル
    • アプリのインストール
    • アプリの許可設定
    • パッケージ型アプリ – アプリの種別や違いの説明もあります
  • アプリの設計
  • アプリの開発
    • アプリマネージャを使用する
    • Firefox の開発者ツール
    • WebAPI
    • Web Activities (アプリ間連携)
    • Simple Push API
    • wiki.mozilla.org の WebAPI ページも参照
  • アプリの公開
    • Firefox Markeplace へのアプリの登録
    • アプリ内課金
    • Marketplace レビュー要件

日本語になっていない記事や英語版に比べて古くなっている記事を見つけたらドキュメント翻訳のガイドラインを参考に日本語化にご協力いただければ幸いです。

Mozilla Hacks ブログ翻訳 | Mozilla Developer Street (modest)


https://dev.mozilla.jp/project/hacksmozillaorg/
Mozilla Hacks Blog では Firefox や Web 技術に関する最新情報が発信されていますが、ここではその記事をコミュニティの有志が翻訳しています。Firefox OS カテゴリの記事はもちろん、それ以外に面白い記事が多いので、未訳の良い記事があったらあなたもここに翻訳を掲載してください。

The Apps Blog | Apps and the Firefox Marketplace


https://blog.mozilla.org/apps/
Firefox Maretplace や Web アプリに関する最新情報が発信される公式ブログです。

アプリレビュアー募集といった Marketplace 運営に関わる記事、HTML5 に関する開発者調査レポートの紹介、Mozilla と協力しているコンテンツパートナーに関する記事などもあります。

Building Firefox OS


http://buildingfirefoxos.com/
Firefox OS 標準アプリ (Gaia アプリ) で用いられているユーザインターフェイスライブラリです。CSS や画像ファイルをダウンロードし、HTML からリンクすることで標準アプリと同じデザインのアプリを簡単に開発できます。もちろん、このライブラリを使わず一般的な Web 開発者向けのフレームワークを使って開発して問題ありません。

au – Fx0


http://au-fx.kddi.com/products/
2014 年 12 月、au から日本初の Firefox OS 商用スマートフォン Fx0 が発売されました。まずはこちらを購入して開発してみましょう。開発用に通信料金無しで端末代金だけで入手したい方は Fx0 の入手方法や格安SIMでの使い方なども参考にしてください。

T2Mobile – Flame

http://www.mozilla.jp/firefox/os/devices/flame/

Firefox OS の開発者向けリファレンス端末 Flame は日本国内でも安心して使えるように TELEC や JATE などの認証を通過して 2014 年 7 月に販売されました。この端末について詳しくは MDN の開発者向け情報ページをご覧ください。

eBay HK – ZTE Open


http://www.ebay.com/itm/331031989534
Firefox OS で最初に商用展開されたローエンド端末 (Single Core, RAM 256MB etc.) である ZTE Open は SIM フリー版が 69.99 ドルで販売されています。日本へはこちらの eBay 香港のページから注文できるので、Firefox OS の開発機が欲しい方はこちらから注文されるのが良いかと思います。
なお、2014 年 2 月の MWC では Firefox OS Flame という若干スペックが向上した (Dual Core, RAM 1GB etc.) 開発者向けのリファレンス端末を提供予定であると発表されているため、そちらの販売開始を気長に待つという選択肢もあります。

Geeksphone


http://www.geeksphone.com/
開発者向けプレビュー端末 Keon および Peak の販売元です。2013 年 4 月に発売されてから入荷後即完売が続き、完全に売り切れました。2014 年 4 月現在では Revolusion という Android だが Boot to Gecko のビルドにも入れ替え可能な端末が販売されています。
