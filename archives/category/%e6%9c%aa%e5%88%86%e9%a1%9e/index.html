今後開催予定のFxOSコミュニティイベント一覧 2015.02版

コメントをどうぞ

みなさんこんにちは。

国内初の商用 Firefox OS 端末 Fx0 に続き、近く Panasonic から Firefox OS 搭載 TV が発売される予定です。この機会に、 Firefox OS という名前を初めて聞いた人も、普段から馴染みがある人も、みんなで集まって Firefox OS のことをより深く知れるイベントが相次いで開催されます。今回はこれらのうち Firefox OS コミュニティ発のものをいくつか紹介させていただきます。

FxOS コードリーディングミートアップ #14（2/25（水） 19:30-21:30/Mozilla Japan オフィス）
https://atnd.org/events/62387
FxOS のコードを読もう！というコミュニティのミートアップです。 FxOS のソースコードを読むだけではなく、 FxOS について気軽に知れる雰囲気で和気あいあいと集まるミートアップです。

Firefox OS アプリハンズオン on オープンソースカンファレンス 2015 Tokyo/Spring（2/28（日） 10:00-16:00/時間中数度開催）
http://firefox-os.doorkeeper.jp/events/20401
オープンソースカンファレンス 2015 Tokyo/Springの会場内で開催される Firefox OS アプリのハンズオンです。アプリ開発経験がない方でも Firefox OS アプリの作り方やノウハウを1から知ることができます。

Firefox OS アプリハンズオン in Nagoya（2/28（日） 13:00-18:00/名古屋市 sharebase.inc）
https://atnd.org/events/62137
名古屋でも Firefox OS アプリのハンズオンが開催されます。カメラアプリの作成など、 Firefox OS ならではのアプリ作りが体験できます。

関東 Firefox OS 勉強会 9th (3/17（火） 19:00-21:00)
2月に引き続き3月も関東 Firefox OS 勉強会が開催されます。スマホではなくデバイスに搭載される Firefox OS の話が集まる予定です。

Firefox OS アプリハンズオン supported by Gunma.web（3/28 13:00-18:00/高崎駅近く あすなろカフェ）
http://gunmaweb.doorkeeper.jp/events/21025
Fx0 の発売を受けて群馬初の Firefox OS アプリハンズオンが行われます。東京に気軽に来れない、という人にもおすすめのイベントです。もちろん高崎線に揺られていって、ついでに観光！というのもおすすめです。

FxOS コードリーディングミートアップ #15（3/18（水） 19:30-22:00/Mozilla Japan オフィス）
FxOS コードリーディングミートアップは毎月開催されます。気軽に、ふらっと立ち寄ってみてください。

Firfox OS アプリハンズオン at 新潟（3月後半予定）
Firefox OS のアプリハンズオンが3月後半に新潟でも開催される予定です。詳しい情報はイベント詳細がまとまり次第改めてお伝えします。

3/21 0day Firefox OS セキュリティセミナー (仮)
Firefox OS を題材にセキュリティセミナーが開催されます。詳しい情報はイベント詳細がまとまり次第改めてお伝えします。

どれも「初めて Firefox OS に触れる人でも楽しめる」イベントとなっています。スケジュールがあうイベントが1つでもあったらぜひ参加を検討してくださいね。
