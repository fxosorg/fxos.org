Firefox OS & Windows 8 アプリ開発ハッカソン

開催日時
2013/08/10 (土) 10:00〜18:00
会場
Mozilla Japan
定員
40
ハッシュタグ
#fxos

詳細ページ＆申し込み

■概要
みなさん、FirefoxOSもWIndows8もHTML/CSSとJavaScript、俗にい うHTML5でアプリ開発が出来る事をご存知でしょうか。このイベントはFirefoxOS、WIndowsストアアプリの開発者コミュニティが合同で主 催し、各OS向けまたは両OS向けアプリの開発をみんなで楽しく行う事が目的です。
開発内容やスキルに制限はなく、どなたでもご参加出来ます。
午前中はOS毎にアプリの作り方に関するセッションを行い、午後からアプリの作成をするのでそれぞれの開発経験が無い方でも大丈夫です。
アプリは一人で作っても良し、参加者同士で協力して作っても良しです。
チーム開発も出来ますのでデザイナーの方でコーディングは難しい方もお気軽にアプリ開発に参加出来ます！！！
（むしろデザインが入る分クオリティが上がるので大歓迎です）
OS指定で個別アプリを作成してもOKですが、
FirefoxOSとWindows8相互プラットフォームで動くアプリが出来ると素敵です★
■参加に必要な物
・FIrefoxブラウザの入っているPC(FirefoxOS)
・VisualStudio2012または2013の入っているWindowsPC(Windows8アプリ)
