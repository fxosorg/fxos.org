中国 Firefox OS 勉強会 2nd

～ リファレンス端末 Flame 国内販売!! ～

セッションで話していただけるかたを募集しています。

会場は未定ですが広島市内を予定しています。
※ 費用：500円ほどを予定しています。(参加人数によって変動しますのでご留意ください。)

開催概要

中国地区での第2回目となる Firefox OS についての勉強会です。
国内初の Firefox OS 開発者向けリファレンス端末 Flame (*1) が発売されます。
TELEC や JATE などの認証も通り安心して使える端末となります。

日時 : 2014/08/30 (土) 13:30-17:00
定員 : 60人
会場 : (公益財団法人) 広島市文化財団 竹屋公民館
住所 : 〒730-0044 広島市中区宝町3番15号
費用 : 500円ほど

(*1) Flame の公式情報は こちら

  • 価格：18,500 円(税別)を予定しています。送料は一部地域を除き無料となります。
  • 販売：株式会社びぎねっとより、Yahoo ショッピングにて販売されます。
  • 時期：7月中となる予定です。
