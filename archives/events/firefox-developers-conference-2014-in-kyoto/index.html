イベント詳細

Firefox Developers Conference 2014 in Kyoto

開催日時
2014年4月19日 13:00 - 19:00
会場
キャンパスプラザ京都
定員
250
ハッシュタグ
#fxdevcon

詳細ページ＆申し込み

Mozilla Japan 主催の Firefox Developers Conference を 4 月 19 日 (土) に京都で開催します。今回のテーマは、オープンな Web 標準技術に基づくアプリケーションプラットフォームとして、今注目の「Firefox OS」。

基調講演には、米国 Mozilla Corporation で Firefox Marketplace / アプリ担当の Vice President を務める Rick Fant が来日するほか、日本における製品展開も見据えつつ Firefox OS の未来をお伝えします。アプリ開発の基本から最新の開発環境のご紹介、国内コンテンツプロバイダーによるトークセッションなど、多彩なゲストをお招きして Firefox OS のすべてを語り尽くします！

http://www.mozilla.jp/events/devcon/2014/kyoto/
