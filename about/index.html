コミュニティについて

Firefox OS コミュニティとは？

Firefox OS コミュニティには、Firefox OS に興味のある有志が集まっています。デザイナーやエンジニアはもちろん、誰もがそれぞれにできることを見つけて Firefox OS に関して日々研究をし、みんなで楽しみながら Firefox OS に貢献しています。

メーリングリスト (Google Group) などオンラインでの情報交換はもちろん、勉強会を始めとしたイベントを Mozilla Japan オフィスのファクトリースペースなどで開催したり、他のコミュニティや企業などとも協力して勉強会、ハンズオン、ハッカソンなども開催しています。

コミュニティの始まりや活動の様子については日経 Linux のインタビュー記事も参考にしてください。


（Mozilla Japan オフィスのファクトリースペース）

どうやって参加するの？

Firefox OS や Web アプリの開発に興味のある方であれば、どなたでも参加していただけます。
Firefox OS に関するイベントの多くは Doorkeeper やメーリングリストで告知されるので、気になるイベントがあれば直接参加してください。お住まいの地域の近くでイベントが開催されない方は、ご自身でイベントを企画していただいたり、開催の提案やリクエストをしてください。

どんなイベントや活動があるの？

セミナー形式の勉強会は Firefox OS 勉強会が東京で 1~2 ヶ月に一度、大阪、名古屋、広島などでも不定期に実施されています。ブラウザのエンジンや Firefox OS の下回りに注力した勉強会としては Gecko 勉強会も不定期に開催されています。

オンライン・オフラインで情報交換を続ける場としては Firefox OS コードリーディングが毎月ほぼ定期的にミートアップ (に加えて懇親会やノマド) をしており、活発に活動を続けています。

ハンズオンやハッカソンについては、京都クリエイティブワークショップが (京都といいつつ東京で) 少人数のハッカソンを１〜３ヶ月に一度程度のペースで、その他にも時々の話題に応じてハンズオンやハッカソンを不定期に実施しています。

製品のローカライズやドキュメントの執筆・翻訳については MDN 翻訳ミートアップが 1〜2 ヶ月に一度程度開催されているほか、ML やオンラインでのやり取りを行っています。Mozillaドキュメント翻訳の始め方 もご覧ください。

新しいイベント情報のお知らせが欲しい方は Doorkeeper やメーリングリストに登録してください。

情報共有

Twitter での情報共有ではハッシュタグ #fxos を使用しています。Firefox OS に関連する話題であれば、ぜひハッシュタグを付けてつぶやいてください。

この Firefox OS コミュニティとあわせて、Mozilla や Web 技術に関する開発者が集まり知識・アイデアの共有やイベントの開催を行っているコミュニティサイト Mozilla Developer Street (通称 modest) もご覧ください。
